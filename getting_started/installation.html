Installation¶

Required dependencies¶

  • Numpy

  • Xarray (0.13.* or later)

  • Scipy

  • rasterio

  • Nose2

  • json-checker

  • netCDF4

  • numba (0.47.* or later)

  • opencv-python

Instructions¶

This package can be install through the following commands:

$ pip install pandora
