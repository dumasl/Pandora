abstract desc(self)[source]¶

Describes the validation method

abstract disparity_checking(self, dataset_ref: xr.Dataset, dataset_sec: xr.Dataset, img_ref: xr.Dataset = None, img_sec: xr.Dataset = None, cv: xr.Dataset = None) → xr.Dataset[source]¶
Determination of occlusions and false matches by performing a consistency check on valid pixels. Update the validity_mask :

  • If out & MSK_PIXEL_OCCLUSION != 0 : Invalid pixel : occluded pixel

  • If out & MSK_PIXEL_MISMATCH != 0 : Invalid pixel : mismatched pixel

Update the measure map: add the disp RL / disp LR distances

Parameters

  • dataset_ref (xarray.Dataset with the variables : - disparity_map 2D xarray.DataArray (row, col) - confidence_measure 3D xarray.DataArray (row, col, indicator) - validity_mask 2D xarray.DataArray (row, col)) – Reference Dataset

  • dataset_sec (xarray.Dataset with the variables : - disparity_map 2D xarray.DataArray (row, col) - confidence_measure 3D xarray.DataArray (row, col, indicator) - validity_mask 2D xarray.DataArray (row, col)) – Secondary Dataset

  • img_ref – reference Datset image

  • img_sec – secondary Dataset image

  • cv – cost_volume Dataset

Returns

the reference dataset, with the bit 8 and 9 of the validity_mask : - If out & MSK_PIXEL_OCCLUSION != 0 : Invalid pixel : occluded pixel - If out & MSK_PIXEL_MISMATCH != 0 : Invalid pixel : mismatched pixel

:rtypexarray.Dataset with the variables :

  • disparity_map 2D xarray.DataArray (row, col)

  • confidence_measure 3D xarray.DataArray (row, col, indicator)

  • validity_mask 2D xarray.DataArray (row, col)
